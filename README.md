# Student Performance Analysis (Pandas)

## Objective
- Analyze students based on their Performanace in Subjext Wise Marks 
- Used Indian Style Marking System(0-100) 
- Analyze the students as TopStudents , WeakStudents ,Students Data 

## Dataset
- student_performance.csv
- Subjects: Maths, Science, English
- Indian-style academic data (0–100 marks)

## Analysis Performed
- Total marks & percentage calculation
- Performance classification
- Ranking & sorting
- Risk identification
- Clean CSV report generation

## Project Structure
- data/      → raw dataset
- src/       → analysis logic
- outputs/   → generated reports

pip install -r requirements.txt
